# XGBoost-Sniper (Legacy V1)

**Status:** `Deprecated`
**Reason:** High Volatility, Failed Consistency Audit.

This document preserves the original V1 algorithm for historical comparison. While it showed promise with a +6.68% ROI, it failed to pass rigorous stress testing (Flat Betting ROI was negative).

### The Flaw
V1 relied too heavily on "buying wins" (betting heavy favorites) and lacked portfolio management. It would bet on any sport with high confidence, leading to massive losses in efficient markets like the NFL.

### The Solution
See the [Main README](../README.md) for the V2 Platinum architecture, which fixed these issues via Regime Filtering and Value Floors.
